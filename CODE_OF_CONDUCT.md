# Code of Conduct

## 目标

GdeiAssistant 社区希望保持尊重、友善、克制、建设性的协作氛围。该准则适用于 Issue、PR、讨论区、文档、社区交流以及与项目相关的协作场景。

## 我们期望的行为

- 尊重不同背景、经验和观点
- 聚焦事实、代码、产品和问题本身展开讨论
- 在反馈中保持清晰、克制和建设性
- 在涉及安全、隐私、学校系统和用户数据的话题上保持谨慎

## 不被接受的行为

- 骚扰、歧视、辱骂、威胁、羞辱或持续性的人身攻击
- 人肉搜索、泄露隐私、传播他人个人信息或鼓励他人这样做
- 故意引战、刷屏、恶意举报、恶意挑衅、钓鱼或持续破坏讨论秩序
- 明知故犯地发布违法违规、侵权、诈骗或高风险内容

## 处理方式

维护者可根据情节采取以下措施：

- 提醒或要求整改
- 删除评论、Issue、讨论或其他不当内容
- 限制互动、关闭讨论、拒绝合并 PR
- 暂时或永久限制参与项目相关社区

对严重涉及安全、隐私、违法线索或持续骚扰的行为，维护者可保留必要记录并依法依规配合处理。
